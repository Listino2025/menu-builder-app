Torna ai Ristoranti
Importa Ristoranti
Formato file CSV richiesto:
Colonna Obbligatorio Formato/Esempio
name Sì Ristorante Centro Roma
address Sì Via Roma 123
city Sì Roma
restaurant_code Sì REST_ROMA_001
postal_code No 00100
latitude No 41.902782
longitude No 12.496366
phone No +39 06 1234567
email No info@restaurant.com
opening_hours No {"monday": {"open": "09:00", "close": "22:00"}}
Note importanti:
  • restaurant_code deve essere univoco
  • latitude e longitude devono essere entrambi forniti per la mappa
  • opening_hours deve essere in formato JSON valido
  • Il file deve avere l'intestazione con i nomi delle colonne
Importa Listini Prezzi
Formato file CSV richiesto:
Colonna Obbligatorio Formato/Esempio
restaurant_code Sì REST_ROMA_001
product_name Sì Big Mac
local_price Sì 8.50
delivery_price Sì 10.00
is_available No true/false/sì/no/1/0
Prerequisiti:
  • I ristoranti devono essere già stati importati o creati
  • I prodotti devono essere già stati creati nel sistema
  • restaurant_code deve corrispondere a un ristorante esistente
  • product_name deve corrispondere esattamente al nome di un prodotto esistente
Guida all'Importazione CSV
Preparazione File CSV
  • Encoding: UTF-8
  • Separatore: Virgola (,)
  • Prima riga: Intestazioni colonne
  • Formati numeri: Punto (.) per decimali
  • Testo: No virgolette se non necessarie
Ordine di Importazione
  1. Prima importa i Ristoranti
  2. Assicurati di avere Prodotti creati
  3. Poi importa i Listini Prezzi
Esempi di Contenuto CSV
Ristoranti (restaurants.csv): 
name,address,city,restaurant_code,latitude,longitude
Ristorante Centro,Via Roma 123,Roma,REST_ROMA_001,41.902782,12.496366
Ristorante Nord,Via Milano 456,Milano,REST_MILANO_001,45.464664,9.188540
Listini Prezzi (listings.csv): 
restaurant_code,product_name,local_price,delivery_price,is_available
REST_ROMA_001,Big Mac,8.50,10.00,true
REST_ROMA_001,Coca Cola,2.50,3.00,true
REST_MILANO_001,Big Mac,9.00,11.00,false
Validazione Errori
  • Il sistema controllerà ogni riga
  • Gli errori saranno mostrati con numero riga
  • Le righe valide saranno importate
  • Le righe con errori saranno ignorate
